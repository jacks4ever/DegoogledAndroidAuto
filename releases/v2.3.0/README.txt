DegoogledAndroidAuto v2.3.0

This release includes important fixes for app recognition and car connectivity issues.

Key improvements:
- Fixed app recognition for apps installed via Aurora Store
- Fixed "Required permission not granted" error when connecting to cars
- Fixed "No installed apps work with this USB accessory" error
- Added support for alternative messaging apps and home automation apps
- Improved USB accessory detection and connection handling

For full details, see RELEASE_NOTES.md

Installation:
1. Download the APK file
2. Enable "Install from Unknown Sources" in your Android settings
3. Install the APK
4. Grant ALL requested permissions when prompted
5. Connect your phone to your car's USB port
6. Tap "Connect to Car" in the app
