DegoogledAndroidAuto v2.2.0

This release includes new messaging integrations for WhatsApp, Microsoft Teams, and Handcent SMS, along with privacy enhancements and bug fixes.

Files included:
- DegoogledAndroidAuto-v2.2.0.apk: The fully degoogled version
- MinimalGoogleAuto-v2.2.0.apk: The minimal Google version for car head unit integration
- RELEASE_NOTES.md: Detailed release notes

Installation instructions and documentation can be found in the main README.md file in the repository.

For support or to report issues, please open an issue on the GitHub repository.
